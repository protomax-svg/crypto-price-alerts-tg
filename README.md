# Crypto Price Alerts Telegram Bot

This is a Telegram bot that sends price alerts for Binance trading symbols(spot).
 You can create multiple alerts and the bot will notify you when the target price is reached.

Installation:
 Create .env in the project directory and add your Telegram bot token there in this format: TELEGRAM_BOT_TOKEN=your_token_here. 
 Install dependencies from requirements.txt. 
 Run the bot with python price_alertsF.py.
